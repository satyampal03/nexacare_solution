# Nexacare Solution - Marketing Website

A modern, high-performance marketing website for Nexacare Solution - a premium Dubai-based digital agency and BPO provider.

## 🚀 Features

- **Modern Tech Stack**: Built with React 19, TypeScript, Vite, and Tailwind CSS
- **Stunning Animations**: Powered by Framer Motion and GSAP
- **3D Visual Effects**: Interactive Three.js particle system
- **Smooth Scrolling**: Implemented with Lenis for buttery-smooth scrolling
- **Fully Responsive**: Optimized for all devices and screen sizes
- **SEO Optimized**: Meta tags and structured data for better search visibility

## 🛠️ Tech Stack

- **Framework**: React 19
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4
- **Animations**: Framer Motion, GSAP
- **3D Graphics**: Three.js
- **Routing**: React Router DOM
- **Build Tool**: Vite
- **Smooth Scroll**: Lenis

## 📦 Installation

1. Clone the repository:
```bash
git clone git@github.com:rakesh15sri-create/nexacare_solution_marketing.git
cd nexacare_solution_marketing
```

2. Install dependencies:
```bash
npm install
```

3. Create environment file:
```bash
cp .env.example .env
```

4. Start the development server:
```bash
npm run dev
```

The application will be available at `http://localhost:3000`

## 🏗️ Build

To build for production:

```bash
npm run build
```

To preview the production build:

```bash
npm run preview
```
